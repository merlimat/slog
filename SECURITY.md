# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly
via [GitHub Security Advisories](https://github.com/merlimat/slog/security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

You should expect an initial response within 72 hours. Once the issue is confirmed, a fix
will be developed and released as soon as possible, depending on the complexity of the
vulnerability.
